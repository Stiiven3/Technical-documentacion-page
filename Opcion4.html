Condicionales

Las condicionales son utiles, con ella podemos evaluar una condicion y si es verdadera podemos hacer que una parte de nuestro codigo se ejecute en el caso de que no sea verdadera no se ejecura.

Tenemos el if anidado:

  • Las sentencias if implementan decisiones que implican una o dos alternativas.
  • Una sentencia if es anidada cuando la sentencia de la rama verdadera o la rama falsa es a su vez una sentencia if..
  • Una sentencia if anidada se puede utilizar para implementar decisiones con varias alternativas o multialternativas.
